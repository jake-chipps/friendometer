Peaks and Potentials


Arduino and Physical Computing

The Friend-O-Meter

This curriculum was designed for Montana State University Peaks and Potentials summer program. Using an Arduino microcontroller, LEDs, and sensors, participants will spend five days creating a DIY touch sensor "friend-o-meter." Each aspect of design and programming is broken down into sub-parts, where module is its own lesson, leading to a functioning computational artifact.

Integrable

This curriculum can be integrated into any subject matter in formal or informal education settings. You just need 750 minutes of instructional time.

Project Based

Each lesson culminates in the construction of a participant-designed computational artifact. Bringing together makerspaces, constructionism, and narrative-based education, this curriculum grounds learning in active, hands-on learning.

For Everyone

This curiculum is a step-by-step process in creating circuits using a microcontroller and developing computational artifacts using input and output devices. This is a great place to start if you have never worked with computing systems or Arduino before. But even those familiar with Arduino may still find value in this project. The goal is to be able to create your own projects after completing this project.
